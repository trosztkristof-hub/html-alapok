METU CSS Kert

A stíluslapok szépsége

Üdvözlünk a Budapesti Metropolitan Egyetem Médiadesign tanszékének CSS gyakorlatán. Ez az oldal azt demonstrálja, hogy kizárólag CSS segítségével milyen vizuális világok teremthetők — miközben a HTML struktúra változatlan marad.

A feladatod egyszerű: formáld meg ezt az oldalt a saját stílusodra. A HTML kódhoz nem nyúlhatsz — kizárólag a style.css fájlt szerkesztheted. Mutasd meg, mire képes a CSS!

Mi ez a feladat?

A CSS Kert egy klasszikus webfejlesztési gyakorlat, amelyet az eredeti CSS Zen Garden ihletett. A lényeg: egyetlen HTML dokumentum, végtelen megjelenési lehetőség. Minden a CSS-en múlik.

Nézz körül az interneten inspirációért, majd alkoss valami egyedit. Legyen bátor, legyen kreatív, legyen a tiéd.

Játékszabályok

  • A HTML fájlt tilos módosítani
  • Csak a style.css fájlban dolgozhatsz
  • Külső képeket használhatsz háttérként
  • Google Fonts betűtípusok szabadon használhatók
  • Az oldal legyen olvasható és használható

Inspiráció

A jó design nem véletlen — tudatos döntések sorozata. Gondolkodj el a következőkön:

Színek

Válassz egy összehangolt színpalettát. Kevesebb néha több.

Tipográfia

A betűtípus hangulatot teremt. Párosíts egy címbetűt egy olvasmányos szövegtípussal.

Elrendezés

Használd ki a teret. A margók és a padding a legjobb barátaid.

Részletek

Kerekített sarkok, árnyékok, átmenetek — a kis dolgok számítanak.
