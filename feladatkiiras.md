## A feladat lényege

Egy előre elkészített HTML oldalt kell megformáznod **kizárólag CSS segítségével**. A HTML kódot nem módosíthatod — csak a `style.css` fájlban dolgozhatsz. Ez a klasszikus „CSS Zen Garden" típusú gyakorlat, amely megtanít arra, hogy a tartalom (HTML) és a megjelenés (CSS) hogyan választható szét.

## Fájlok

| Fájl | Leírás | Szerkeszthető? |
|------|--------|:-:|
| `index.html` | Az oldal struktúrája és tartalma | ❌ NEM |
| `style.css` | Az oldal megjelenése — ez a te vásznod | ✅ IGEN |

## Lépések

1. **Töltsd le** mindkét fájlt és nyisd meg a kódszerkesztődben (VS Code ajánlott)
2. **Nyisd meg** az `index.html`-t böngészőben — így látod az alap (formázatlan) állapotot
3. **Szerkeszd** a `style.css` fájlt — minden változtatás után frissíts a böngészőben (Ctrl+R / Cmd+R)
4. **Kísérletezz** bátran — a CSS-ben nem lehet „elrontani" a HTML-t

## Minimum elvárások (alap szint)

Legalább az alábbi CSS tulajdonságokat használd:

- [ ] **Háttérszín** (`background-color`) az oldalon és/vagy egyes szekciókban
- [ ] **Betűtípus** (`font-family`) — legalább egy Google Font importálása
- [ ] **Betűméret** (`font-size`) — hierarchia a címek és szövegek között
- [ ] **Színek** (`color`) — tudatos színválasztás, legalább 3 szín
- [ ] **Box model** — `margin` és `padding` használata a térközök szabályozásához
- [ ] **Szegélyek** (`border`, `border-radius`) — legalább egy elemen
- [ ] **Flexbox** — az `.inspiracio-racsozat` vagy `.tech-lista` flexbox elrendezése

## Bónusz (extra pontért)

- [ ] `box-shadow` használata
- [ ] `hover` effekt legalább egy elemen
- [ ] `transition` animáció
- [ ] Háttérkép (`background-image`) használata
- [ ] Az oldalsáv (`aside`) és a fő tartalom (`main`) flexbox-szal egymás mellé rendezése
- [ ] Google Fonts párosítás (egy cím + egy szöveg betűtípus)
- [ ] Összehangolt színpaletta (pl. coolors.co segítségével)
